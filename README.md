# To-Do Game

Welcome to the To-Do Game, an interactive task management application that gamifies your daily to-do list. This game aims to make completing tasks more fun and engaging.

## Features

- Add, edit, and delete tasks
- Mark tasks as completed
- Earn points for completing tasks
- Progress through levels based on points
- Interactive UI with animations
- Save tasks and progress locally

## Installation

1. Clone the repository:
    ```sh
    git clone https://github.com/yourusername/todo-game.git
    ```

2. Navigate to the project directory:
    ```sh
    cd todo-game
    ```

3. Open `index.html` in your web browser to start the application.

## Usage

1. **Adding a Task**: Enter a task in the input field and click the "Add Task" button.
2. **Editing a Task**: Click on the task text to edit it, and press "Enter" to save changes.
3. **Deleting a Task**: Click the trash icon next to the task you want to delete.
4. **Completing a Task**: Click the checkbox next to the task to mark it as completed and earn points.
5. **Viewing Points and Levels**: Your points and current level are displayed at the top of the application. Completing tasks will increase your points and level.
